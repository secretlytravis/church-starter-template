# Church Starter Template

A static church website starter that another church can copy, customize, and deploy quickly.

## Structure

- `config/` contains editable starter values.
- `index.html` and `404.html` stay at the root for hosting.
- `pages/` contains standard interior pages.
- `testimonies/` contains individual testimony pages.
- `assets/` contains shared images, icons, CSS, JavaScript, data, and the web manifest.
- `netlify/functions/` contains the sermon feed function.
- `tools/` contains local development helpers.

Start with `SETUP.md`, then see `PROJECT_OVERVIEW.md` for maintenance notes, deployment details, and the sermon feed explanation.
